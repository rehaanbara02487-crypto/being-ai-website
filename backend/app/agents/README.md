# Agents

Specialist LLM agents. See `docs/AGENTS.md`.

| Module | Agent |
|--------|-------|
| `planner.py` | Requirements → structured Plan |
| `coder.py` | File create/edit per plan step |
| `terminal.py` | Shell command execution |
| `debug.py` | Error analysis → fix actions |
| `github.py` | Git + GitHub remote |
| `memory.py` | Persist project learnings |
| `base.py` | Shared LLM config, prompts |
